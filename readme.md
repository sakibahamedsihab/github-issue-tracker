2️⃣ What is the spread operator (...)?
    spread operator is like spreading the value of array one by one. it is used to separate the values inside an array also for creating new array and object
3️⃣ What is the difference between map(), filter(), and forEach()?
    map -> it iterate all the elements of the array and does action and then return a new array.
    filter -> it filter out value from the given array according to particular logic and then return a new array
    forEarch -> unlike other two it does not return a new array rather it performs it's logic within the provided array
